## 1. Obtener rango por DirecciÃ³n

```javascript
await Excel.run(async (context) => {
    let sheet = context.workbook.worksheets.getItem("Sample");
    
    let range = sheet.getRange("B2:C5");
    range.load("values");
    await context.sync();
    
    console.log(`The address of the range B2:C5 is "${range.values}"`);
});
```
